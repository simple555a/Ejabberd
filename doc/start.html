By default, the ejabberd Web Admin interface is enabled. You should be able to connect to the admin interface and log in with the user admin@xxw-PC and the password you defined during the installation.

Users can connect to your server 'xxw-PC' with any Jabber/XMPP client. Direct registration from the user client is enabled by default.

If you are an advanced user, you can check the ejabberd service logs or the Erlang/OTP system logs to check how things are going.

The name of the current ejabberd node is ejabberd@xxw-PC.

Refer to ejabberd Installation and Operation Guide for documentation about ejabberd.
